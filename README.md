# Survey Form Project

## Description
This is a beginner-level Survey Form project created using HTML, CSS, and JavaScript.

The form allows users to:
- Enter Name
- Enter Email
- Enter Phone Number
- Enter Message
- Submit details
- View submitted responses below the form

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Simple user interface
- Responsive form layout
- Displays submitted data dynamically

## How to Run
1. Download the project
2. Open in VS Code
3. Open index.html using Live Server
